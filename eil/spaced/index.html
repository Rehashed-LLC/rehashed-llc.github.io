EIL Space Explorer

Real-time visualization of cross-chain transactions across EVM chains.

Timeline
  • Starts in live mode showing real-time transactions
  • Click "Time Machine" to replay historical data
  • Select time range: 1h, 4h, 12h, or 24h
  • After replay, returns to live data stream
Navigation
  • Click on any chain node to focus camera
  • Drag to rotate the view
  • Scroll to zoom in/out
  • Toggle "Follow" to auto-track transactions
Visuals
  • Each glowing node represents an EVM chain
  • Nodes pulse and grow when sending/receiving
  • Particle streams show cross-chain transfers
  • Colors indicate source and destination chains
